# Workflows Included

## Lead Scraper → Google Sheets CRM → Auto Email  
**Folder:** `workflows/lead_scraper_crm_followup.json`  
**Doc:** `docs/lead_scraper_overview.md`

**Problem Solved:**  
Businesses waste hours manually collecting emails from websites and sending outreach messages.

**Solution:**  
Automatically scrape leads → extract emails → store in Google Sheets → send personalized emails via Gmail.

---

## AI Customer Support Auto-Responder  
**Folder:** `workflows/ai_auto_support_responder.json`  
**Doc:** `docs/ai_auto_support_overview.md`

**Problem Solved:**  
Teams receive too many customer emails and respond slowly.

**Solution:**  
Automatically read incoming Gmail messages → summarize → generate a professional AI reply → send instantly.

---

## Daily Revenue Report Generator  
**Folder:** `workflows/daily_revenue_report.json`  
**Doc:** `docs/daily_revenue_overview.md`

**Problem Solved:**  
Founders want daily revenue visibility without logging into dashboards.

**Solution:**  
Stripe payment webhook → Google Sheets logging → Daily revenue summary email.
